e-ReZeta
========

- Build Status: [![Build Status](https://travis-ci.org/paolocarrasco/rz.png)](https://travis-ci.org/paolocarrasco/rz)

Requirements
------------

Install:
- Node 6.4
- Gulp (execute `npm install --global gulp-cli`

Install dependencies:
- Execute in the project directory:
```
npm install
```

Execution
---------

Execute in the project directory:

```
npm run serve
```
